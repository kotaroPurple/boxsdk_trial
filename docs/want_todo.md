
# 本プロジェクトで実現したい BOX SDK (API) の利用

## 背景

- 実験のセンサデータ 1分ごとに csv ファイルとしてローカル PC のディレクトリに保存される
- 実験後にセンサデータを共有ディレクトリである BOX にアップロードしている
- アップロードを手動のドラッグ&ドロップで行っているため人為的ミスが起こる

## 目的

- SDK (API) を利用し自動でアップロードできるか確認する
- 複数人が使うことが想定されるため JWT 認証を使いたい

## 環境

- 開発者は普段 Python を使っている
- GitHub を使っており、.env ファイルは gitignore したい
- MVP 思想で作る
